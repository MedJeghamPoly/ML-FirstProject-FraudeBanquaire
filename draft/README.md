# Dossier `draft/`

Ce répertoire regroupe des **versions intermédiaires** et des **expérimentations** réalisées avant la structuration finale du projet (notebooks monolithiques, exécutions de test, brouillons).

**Contenu typique :**

- Ancienne version **tout-en-un** du pipeline (`fraud_detection_pipeline.ipynb`) conservée pour historique.
- Copie exécutée (`*_executed.ipynb`) utile pour relecture hors session sans relancer le calcul.
- Brouillons de texte ou exports intermédiaires (facultatif).

Les livrables **officiels** pour la remise sont dans **`notebooks/`**, **`src/`**, **`README.md`**, **`requirements.txt`**, et le rapport à la racine — pas dans `draft/`.
